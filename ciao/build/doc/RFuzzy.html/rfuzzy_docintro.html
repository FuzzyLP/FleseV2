rfuzzy

This package implements an extension of Prolog to deal with uncertainty. Consult the following papers for detailed information about the semantics:

  • [MHPCS10]
  • [VAS10]
  • [ASV10]
  • [SMHP09]
  • [MHPS09]
  • [PSM09]
  • [PMS08]

Usage and interface

  • Library usage:
    :- use_package(rfuzzy). or :- module(...,...,[rfuzzy]).
  • Implicit imports:
    • Internal (engine) modules:
      term_basic, arithmetic, atomic_basic, basiccontrol, exceptions, term_compare, term_typing, debugger_support, basic_props.
    • Packages:
      prelude, initial, condcomp, assertions, assertions/assertions_basic, regtypes.
